# ParkSmart Navigation App

Aplicación móvil desarrollada con React Native + Expo.

---

# 🚀 Instalación

## 1) Clonar repositorio

```bash
git clone TU_REPO
````

---

## 2) Entrar a la carpeta

```bash
cd ParkSmart
```

---

## 3) Instalar dependencias

```bash
npm install
```

---

## 4) Instalar dependencias de Expo

```bash
npx expo install
```

---

## 5) Iniciar proyecto

```bash
npx expo start
```

---

# 📱 Ejecutar en celular

1. Descargar Expo Go
2. Escanear QR
3. Esperar compilación

---
